# 🎓 Placement Management System

Welcome to the **Placement Management System**! This comprehensive web application streamlines and enhances the placement process for final-year college students, simplifying data validation and KYC procedures.

## 📦 Project Overview

The **Placement Management System** is designed to streamline the placement process for final-year college students. Key features include:

- **🔐 User Authentication**: Users must log in using their college ID to access the platform.
- **👩‍🎓 Student Features**:
  - **📅 View Latest Drives and Register** for opportunities.
  - **💬 Access Chatroom** for real-time queries and support.
  - **📝 Submit KYC and SPF Forms** with automatic verification of academic marks.
  - **📊 Check Academic Performance** and eligibility for various drives.

- **👨‍🏫 FPC Features**:
  - **✅ Verify** and approve student data submissions.
  - **🔄 Respond** to student queries in the chatroom.

- **👩‍🏫 Placement Trainer Features**:
  - **📈 Manage** and update student marks across various subjects and skills.
  - **💡 Provide support** in the chatroom for doubts and clarifications.

- **👔 Placement Officer Features**:
  - **📝 Create** and publish detailed placement drives with eligibility criteria, packages, and registration links.
  - **📋 Manage** all drive-related activities, including updating and deleting old drives.

- **👨‍💼 Admin Features**:
  - **🔍 Comprehensive access** to all functionalities, including user role management and oversight of the entire system.

This project enhances communication and efficiency, ensuring students and recruiters have up-to-date information and support throughout the placement process.

### 5. **Admin Role**
   - **🔍 Comprehensive Access**: The admin has full access to all functionalities within the Placement Management System, allowing oversight of all user roles and activities.
   - **🛠️ User Management**: Admins can create, update, and delete user accounts, ensuring that access is appropriately granted based on roles and responsibilities.
   - **📊 Data Integrity and Reporting**: They monitor system performance, generate reports on placements, and maintain the overall integrity of the platform, ensuring accurate and up-to-date information for all users.

## 🛠️ Technologies Used

- **⚛️ ReactJS**: Frontend framework for building user interfaces.
- **🐢 NodeJS**: Backend JavaScript runtime for server-side operations.
- **🗄️ MongoDB**: NoSQL database for data storage.
- **🧱 Bootstrap**: Frontend framework for responsive design.
- **🌈 Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **⚙️ ExpressJS**: Web framework for Node.js.
